Cargar y Mostrar Im√°genes

Primeros pasos con OpenCV en Python.

Cargar una Imagen

OpenCV hace muy sencilla la carga de im√°genes desde disco. La funci√≥n cv2.imread() lee la imagen y la devuelve como un array de NumPy.

# Cargar imagen en escala de grises

Nota importante: OpenCV carga las im√°genes en formato BGR (Blue-Green-Red), no RGB como estamos acostumbrados. Tenlo en cuenta al trabajar con otras librer√≠as como Matplotlib.

Mostrar una Imagen

Para visualizar una imagen en una ventana, usamos cv2.imshow():

import cv2

img = cv2.imread('imagen.jpg')

# Mostrar la imagen
cv2.imshow('Nombre de la Ventana', img)

# Esperar hasta que se presione una tecla
cv2.waitKey(0)

# Cerrar todas las ventanas
cv2.destroyAllWindows()
